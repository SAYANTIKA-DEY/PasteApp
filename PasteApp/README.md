# Paste App

A simple **Paste Management Application** built using **React** and **Tailwind CSS** where users can create, view, update, and delete text pastes.

Live Demo: https://pasteapp-kohl.vercel.app/

## 🚀 Features

* Create new paste
* View all pastes
* Update existing paste
* Delete paste
* Responsive UI using Tailwind CSS
* Data stored in browser local storage

## 🛠 Tech Stack

* React
* Tailwind CSS
* Vite
* JavaScript

## ⚙️ Installation

Clone the repository

```
git clone https://github.com/your-username/paste-app.git
```

Go to the project folder

```
cd paste-app
```

Install dependencies

```
npm install
```

Run the development server

```
npm run dev
```
